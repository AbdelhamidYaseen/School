Basic Layout

A HTML page is build up out of various sections there are 2 main parts the "head" and the "body"

DOCTYPE

DOCTYPE refrences the version of HTML we are working in currently

html

References HTML in general

head

head is the upper part of the page, it contains all invisible information and the title that will be displayed in the search bar

body

body is the middle part this contains all the information for the general buildup of the page.

footer

the lowest part of the page, contains permanent information that will be carried between pages
